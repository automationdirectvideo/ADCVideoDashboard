ADC Video Dashboard Shortcuts

These are the available keyboard shortcuts for using the Video Dashboard.
Press Ctrl+P or click here to print this page.

General
F2 Login
Space Pause Dashboard Scrolling
← or → Advance Dashboard Screen
1 , 2 , 3 ,..., E , F Go to Screen
Ctrl + W Exit YouTube Video Window
V Swap Category Performance Charts
Other
Alt + H Go to Main Dashboards
Alt + U Update Dashboards
Menu
M Open/Close Menu
1 - 4 Select Menu Item
↑ or ↓ Select Menu Item
Enter Go to Selected Page
